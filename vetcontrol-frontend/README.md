# VetControl
Sistema de gestión de turnos para veterinarias desarrollado en .NET como Trabajo Integrador Final. Permite administrar turnos, clientes y mascotas de forma centralizada.

## Tecnologías

- React
- Vite
- ASP.NET Core Web API
- SQL Server
